It takes in a password and locks the turret, adding a layer of security to it! It also has a green status light that tells you if the turret is unlocked or not, along with the lcd display. Everything is conrolled using a Raspberry Pi Pico W and parts from the hack club bin. The slider turns on the laser when it is unlocked, which is replaced with a red led in the circuit.
